# اسپات‌داون (SPOTDOWN)

یک دانلودر موسیقی اسپاتیفای با تم سایبرپانک و رابط کاربری وب نئونی.

## ویژگی‌ها

- دانلود آهنگ‌ها، آلبوم‌ها و پلی‌لیست‌های اسپاتیفای به صورت MP3
- درج کامل متادیتا (عنوان، هنرمند، آلبوم، کاور)
- انتخاب بیت‌ریت ۱۹۲ یا ۳۲۰ kbps
- رابط کاربری واکنش‌گرا با تم نئون/سایبرپانک
- نمایش پیشرفت دانلود به صورت لحظه‌ای با SSE
- انتخاب پوشه خروجی
- پشتیبانی از پلی‌لیست و آلبوم با واکشی نامحدود
- درج کاور آرت با وضوح بالا (۶۴۰x۶۴۰)
- گزینه درج متن آهنگ (lyrics) در فایل

## نیازمندی‌ها

- Python 3.10+
- FFmpeg (باید در PATH باشد)
- yt-dlp

## نصب و راه‌اندازی

۱. کلون کردن مخزن:
   ```bash
   git clone https://github.com/mahdim43/spotify-downloader.git
   cd spotify-downloader
   ```

۲. ایجاد محیط مجازی:
   ```bash
   python -m venv venv
   venv\Scripts\activate  # ویندوز
   source venv/bin/activate  # لینوکس/مک
   ```

۳. نصب وابستگی‌ها:
   ```bash
   pip install -r requirements.txt
   ```

۴. (اختیاری) ایجاد فایل `.env` برای اعتبارنامه API اسپاتیفای:
   ```
   SPOTIFY_CLIENT_ID=your_id
   SPOTIFY_CLIENT_SECRET=your_secret
   ```

۵. اجرای سرور:
   ```bash
   python -m uvicorn main:app --host 127.0.0.1 --port 8000
   ```

۶. باز کردن http://localhost:8000 در مرورگر.

## نحوه استفاده

۱. لینک آهنگ/آلبوم/پلی‌لیست اسپاتیفای را وارد کنید
۲. بیت‌ریت مورد نظر را انتخاب کنید (۱۹۲ یا ۳۲۰ kbps)
۳. در صورت تمایل گزینه درج متن آهنگ را فعال کنید
۴. روی دکمه **EXEC** کلیک کنید
۵. پیشرفت دانلود به صورت لحظه‌ای نمایش داده می‌شود
۶. پس از اتمام، فایل‌های موفق با علامت سبز و ناموفق با علامت قرمز نمایش داده می‌شوند

## تکنولوژی‌ها

- **بک‌اند**: Python, FastAPI, uvicorn
- **فرانت‌اند**: Vanilla JS, CSS3 (تم نئون)
- **دانلود**: yt-dlp + FFmpeg
- **متادیتا**: Mutagen (ID3 tags)
- **متن آهنگ**: lrclib.net + lyrics.ovh

## نکات مهم

- برای نمایش کاور در ویندوز، تگ‌ها با فرمت ID3v2.3 ذخیره می‌شوند
- جستجوی یوتیوب با اولویت نسخه‌های صوتی (audio-only) انجام می‌شود
- نام فایل‌ها پس از دانلود از پسوندهای یوتیوب پاکسازی می‌شود
